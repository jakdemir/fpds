# Functional Program Design in Scala

Coursera course assignments for Functional Program Design in Scala (EPFL).

## Overview

This repository contains my solutions for the Functional Program Design course, part of the Scala specialization on Coursera by École Polytechnique Fédérale de Lausanne (EPFL).

## Topics Covered

- For-comprehensions and monads
- Lazy evaluation
- Functional reactive programming
- Reactive streams

## Assignments

- `calculator/` - Spreadsheet calculator
- `quickcheck/` - Property-based testing
- `streams/` - Lazy evaluation with streams

## Technologies

- Scala
- SBT (Scala Build Tool)
- ScalaCheck

## Running

```bash
cd <assignment>
sbt test
```
